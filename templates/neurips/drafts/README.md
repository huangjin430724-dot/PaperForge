# Drafts

This folder stores source drafts and intermediate writing files.

- source_zh.md: preserved Chinese source manuscript.
- working_zh2en.tex: optional temporary LaTeX conversion workspace.

Recommended workflow:
1. Paste the Chinese manuscript into source_zh.md.
2. In Agent, choose 鈥滀腑璇戣嫳 LaTeX鈥?
3. Review the generated English LaTeX diff.
4. Apply the changes to the main TeX file.
5. Compile PDF.
